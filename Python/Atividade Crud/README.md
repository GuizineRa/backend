# CRUD de Personagens em Python

##  Descrição
Este projeto é um sistema simples de CRUD (Create, Read, Update, Delete) feito em Python, que permite gerenciar personagens organizados por classes.

Os dados são armazenados em um arquivo JSON, simulando um banco de dados.

---

##  Funcionalidades

- Adicionar personagem  
- Listar personagens  
- Atualizar personagem  
- Excluir personagem  

---

##  Estrutura dos dados

Os personagens são salvos no arquivo `personagem.json` no seguinte formato:

```json
{
    "dps": [],
    "tanques": [],
    "suporte": []
}
